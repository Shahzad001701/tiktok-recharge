GitHub Pages - Upload Instructions (Urdu)

1) GitHub پر لاگ ان کریں اور نیا public repository بنائیں:
   Example name: tiktok-recharge

2) اس ZIP کے اندر موجود فائلیں upload کریں:
   - index.html
   - style.css
   - script.js
   - README.txt

3) بہتر ہے فائل کا نام 'index.html' ہی رہے — یہ GitHub Pages کی default فائل ہے.

4) GitHub repo میں:
   Settings -> Pages -> Source -> Choose 'main' branch and folder '/ (root)' -> Save

5) چند سیکنڈ کے بعد آپ کی site اس URL پر live ہوگی:
   https://YourUserName.github.io/REPOSITORY_NAME

نوٹس:
- یہ مکمل طور پر UI-only ہے۔ کوئی حقیقی پیمنٹ یا backend calls نہیں ہیں۔
- اگر آپ چاہیں تو میں تصاویر (logo) یا اضافی CSS animations شامل کر دوں۔
